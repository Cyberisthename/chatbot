# Icons

Place your Cortana-themed icons here.

Required files:

- `tray-icon.png` – 32x32 transparent icon for the system tray
- `cortana.ico` – 256x256 Windows ICO for application packaging
- `cortana.png` – 512x512 PNG for marketing/screenshots (optional)

You can export assets from your Cortana APK or design new ones with Fluent Design guidelines.
