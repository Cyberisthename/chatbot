# Jarvis-2v-main Extracted Contents

This directory is intended to contain the contents of the `Jarvis-2v-main.zip` archive.

The archive is not included in the repository. To populate this folder:

1. Place `Jarvis-2v-main.zip` at the project root (same level as this folder).
2. Run the provided extraction script:

   ```bash
   ./extract_jarvis_2v.sh
   ```

3. After extraction, the contents of the archive will be available here.

> **Note:** If the archive is already extracted, you can remove this README.
